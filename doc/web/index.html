libunittest C++ library ↓latest version

libunittest is a portable C++ library for unit testing making use of C++11. libunittest got inspirations from CppUnit, JUnit, and PyUnit and attempts to combine their advantages in a single testing framework for C++. By relying on the latest C++11 standard, heavy use of templates, and minimal use of macros, a portable, feature-rich, and easy-to-use library has been developed.

Features include:

  • unit and integration testing in C++
  • follows the xUnit test patterns
  • formatted XML output of the test results
  • comes with rich assertions and is easily extendable
  • option to run specific tests without recompiling
  • option to turn off exception handling for debugging purposes
  • supports setting timeouts for specific tests or all tests
  • supports test contexts that are shared across multiple tests
  • functionality to easily produce random values and containers
  • portable across various operating systems
  • tests can be run in parallel
  • macros only for registering test runs
  • comes with tests and examples
