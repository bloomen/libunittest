Installation

Prerequisites

Everything that libunittest needs is a C++11 compliant compiler on a UNIX-like operating system and an implementation of the C++11 standard library. libunittest has been successfully tested on Linux, MacOS, and Cygwin/Windows using g++4.7 and clang++3.2.

Install From Source

It is easy to install libunittest from source. First download a source distribution. Now unpack the tar-ball you've just downloaded and enter the resulting directory: tar xfz libunittest-XXX.tar.gz
cd libunittest-XXX libunittest can now be compiled on your machine: make [CXX=compiler] where CXX defaults to g++. When using clang++ in version 3.2, type: make CXX='clang++ -stdlib=libc++ -U__STRICT_ANSI__' Once the compilation is finished the library can be installed by: make install [INSTALLDIR=/path/to/install] where INSTALLDIR defaults to /usr/local.

Install From Debian Package

The libunittest debian packages were compiled with g++ and are currently available for x86 architectures. After downloading, the libunittest debian package can be installed by: sudo dpkg -i libunittest-dev_XXX_i386.deb
