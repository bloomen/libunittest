Installation

Prerequisites

Everything that libunittest needs is a C++11 compliant compiler and an implementation of the C++11 standard library. libunittest has been successfully tested on Linux, MacOSX, and Cygwin/Windows using g++4.7 and clang++3.2. libunittest is expected to compile with VS2013.

Install from source

First download a source distribution, unpack the tar-ball you've just downloaded, and enter the resulting directory: tar xfz libunittest-XXX.tar.gz
cd libunittest-XXX where XXX stands for some version string. libunittest can now be compiled: make [CXX=compiler] where CXX defaults to g++. When using clang++, type: make CXX='clang++ -stdlib=libc++' Installation instructions for libc++ can be found here. Once the compilation is finished libunittest can be installed: make install [INSTALLDIR=/path/to/install] where INSTALLDIR defaults to /usr/local.

Install from debian package

The libunittest debian packages where compiled using g++ and are currently available for intel x86 architectures. After downloading, the libunittest debian package can be installed: dpkg -i libunittest-dev_XXX_i386.deb where XXX stands for some version string.
